Middleware & context

Patties middleware is an async chain around a route handler.

Middleware shape

type Middleware = (
  req: Request,
  ctx: PattiesContext,
  next: () => Promise<Response>,
) => Promise<Response>;

Use defineMiddleware() for authoring and compose() to wrap a handler.

Context values

PattiesContext includes:

  • params
  • cookies
  • env
  • vars
  • url
  • optional aiContext
  • optional csrf
  • helpers for json(), html(), and redirect()

Cookie behavior

Patties reads Bun's request cookie map when available. For fallback paths where that map is missing, it synthesizes cookies from the Cookie header so middleware can still work consistently.

Response helpers

The built-in context helpers reduce boilerplate for common response types:

  • ctx.json(body, init?)
  • ctx.html(body, init?)
  • ctx.redirect(to, status?)
