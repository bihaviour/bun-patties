description: Production build flow and flags.
---

`patties build` creates a production bundle and prints a summary of the output.

## Behavior

- Loads project config and optional target overrides
- Runs the framework build pipeline
- Reports output directory, server entry size, asset sizes, and route counts
- Regenerates the root `AGENTS.md` file after a successful build

## Flags

- `--target <bun|edge>`
- `--out <path>`
- `--app <path>`
- `--compile`
- `--dev`
